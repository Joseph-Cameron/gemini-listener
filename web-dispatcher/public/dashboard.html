EVENT PARSING RULES

The event string is parsed by a mock AI function. These rules map input to device commands:

Input PatternIntensityReason
$1-$5 + "tip"2 (low)$N tip
$6-$15 + "tip"8 (medium)$N tip
$16+ + "tip"15 (high)$N tip
"subscribe" / "follow"3new subscriber
"gift" / "donation"10gift received
anything else1unrecognized

Intensity range: 0 (off) to 20 (max). All commands are Vibrate.

FIREWALL NOTES

If another AI on this machine cannot reach localhost:3000:

  • Check Windows Firewall: allow Node.js inbound on TCP 3000
  • Try 127.0.0.1:3000 instead of localhost
  • If behind a corporate proxy, bypass for localhost in proxy settings
  • The dashboard itself can send events via the test form above
  • To expose externally: use ngrok http 3000 or port-forward
